JSX란?
: 자바스크립트와 HTML 태그를 섞어 사용하는 문법
- 자바스크립트의 확장 문법

JSX와 자바스크립트 표현식
- 원시 자료형 이외의 객체자료형(배열, 함수 포함)은 지원하지 않는다.

JSX 문법에서 지켜야 할 것들
1) 닫힘 규칙
- JSX의 모든 태그는 여는 태그가 있으면 반드시 닫는 태그도 있어야 함
- HTML 태그 중 <img>, <input>은 닫힘 태그 없이도 사용가능하나 JSX에서는 이를 허용하지 않음
반드시 <img />, <input />과 같이 닫힘 태그 사용

2) 최상위 태그 규칙
: JSX가 반환하는 모든 태그는 반드시 최상위 태그로 감싸야 함
HTML 태그를 최상위 태그로 사용하지 않으려면 <React.Fragment> 혹은 빈태그 '<></>'사용

3) 조건부 렌더링
: 리액트 컴포넌트가 조건식의 결과에 따라 각기 다른 값을 페이지에 렌더링. ex) 페이스북 좋아요 버튼

조건부 렌더링에는 삼항 연산자를 이용한 조건부 렌더링과 조건문을 이용한 조건부 렌더링이 있다.
삼항 연산자는 코드가 매우 간결하지만, 자주 사용할 경우 가독성을 해칠 우려가 있으며, 다중 조건을 작성하기 어렵다.
조건문은 가독성은 좋으나 기본적으로 작성해야 할 코드가 많고, 중복 코드가 발생할 우려가 있다.

결론: 처한 상황에 맞게 적절히 선택해서 써라
